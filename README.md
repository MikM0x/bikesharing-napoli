# Bike Sharing Napoli - Progressive Web App 🚲

This project is a Progressive Web App (PWA) for a Bike Sharing service in the city of Naples. It was developed for the "Tecnologie Web" exam at Università degli Studi di Napoli Parthenope.

## Proposal Documents
The initial proposal documents (RAD, SDD, UML diagrams, and presentation) are available in the root of this repository.

## Features
* **Interactive Map:** View available bike stations in Naples using Leaflet.
* **User System:** Register and login to the platform.
* **Rent a Bike:** Search for available bikes and start a rental.
* **PWA Enabled:** Can be installed on mobile devices and works with unstable network conditions.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript, Leaflet, AJAX/jQuery
* **Backend:** Python, Flask
* **Database:** MongoDB
* **Deployment:** Docker
